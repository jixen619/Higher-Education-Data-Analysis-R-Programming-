🎓 Higher Education Data Analysis (R)
📊 Project Overview

This project performs data analysis on higher education datasets to explore the distribution of Science and Non-Science subjects across universities.
The goal is to understand patterns in subject offerings and compare academic categories between institutions using R programming and data visualization techniques.

This project demonstrates data cleaning, exploratory data analysis (EDA), and visualization skills, which are essential for Data Analyst and Business Intelligence roles.

🎯 Objectives

Analyze university data to identify subject distribution patterns.

Compare Science vs Non-Science subject offerings across universities.

Perform data preprocessing and transformation.

Visualize insights using data visualization techniques.

🛠️ Tools & Technologies

R Programming

dplyr – data manipulation

tidyr – data transformation

ggplot2 – data visualization

📂 Dataset

The dataset contains information about higher education providers and subject categories.
It includes data used to analyze academic program distributions across institutions.

🔍 Data Analysis Workflow
1️⃣ Data Cleaning

Removed inconsistencies and prepared the dataset for analysis.

Structured the data for easier processing.

2️⃣ Data Transformation

Converted the dataset into long format for analysis.

Categorized subjects into Science and Non-Science groups.

3️⃣ Exploratory Data Analysis (EDA)

Aggregated subject counts by university and subject category.

Compared the distribution of subjects across institutions.

4️⃣ Data Visualization

Created bar charts and graphical visualizations using ggplot2.

Highlighted differences in academic subject distributions between universities.

📈 Key Insights

Universities vary in the number of science and non-science programs offered.

Some institutions show a higher concentration of science-related subjects, while others focus more on non-science disciplines.

Visualizations help clearly communicate differences in academic offerings.
